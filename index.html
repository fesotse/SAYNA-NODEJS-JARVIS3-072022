A propos

Cet API permet de lier les inputs effectués par l'utilisateur avec la base de donnée sur l'application Jarvis
La première chose à faire est de connecter l'API à la base de donnée Jarvis (s'il existe). dans le cas contraire, la base de donnée doit d'abord être créée pour permettre la connection à mysql
Ensuite les manipulations sur la base de donnée peut s'opérer après la création des schemas et tables
Pour l'utilisation de l'API par le client: la première chose à faire est de créer un compte en s'inscrivant.
Ensuit l'utilisateur définira par la suite son domicile, les pièces et les différents appareils

Ainsi les routes de l'API sont :

  • "/" : cette page
  • "/login" : permet d'authentifier un utilisateur
  • "/register" : permet d'inscrire un utilisateur
  • "/invitation" : permet d'inviter un utilisateur
  • "/object" : changement d'un objet
